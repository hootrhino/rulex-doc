# 1. 消息转发

# 数据转MQTT Server

# 函数定义

函数名 参数 返回值 说明
DataToMqtt data:string error:string 数据转发到另一个 MQTT Server

# 示例

首先新建好资源，然后编写脚本:

-- Success
function Success()
    print("success")
end
-- Failed
function Failed(error)
    print("failed:", error)
end

-- Actions
Actions = {
    function(data)
        rulexlib:DataToMqtt('UUID', data)
        return true, data
    end
}
